# Instagram Growth Analytics System

An end-to-end growth analytics project simulating Instagram influencer and campaign performance.

## 🚀 Project Overview

This project analyzes how engagement metrics impact purchase behavior.  
It includes data generation, feature engineering, statistical A/B testing, predictive modeling, and an interactive dashboard.

---

## 📊 Key Features

- Synthetic influencer engagement dataset generation
- Engagement score engineering
- A/B testing using statistical hypothesis testing (Z-test)
- Purchase prediction using Logistic Regression
- Interactive Streamlit dashboard for campaign insights

---

## 🧠 Business Objective

To evaluate how engagement signals (likes, comments, shares, saves, CTR) influence conversion rates and identify key drivers of purchase behavior.

---

## 📈 A/B Testing

Segmented influencers into low and high engagement groups and conducted hypothesis testing to evaluate statistically significant differences in conversion performance.

---

## 🤖 Machine Learning

Built a Logistic Regression model to predict purchase probability and determine the most influential engagement features.

---

## 🖥️ Dashboard

The Streamlit dashboard provides:

- KPI metrics (CTR, Conversion Rate, Engagement Score)
- Engagement distribution visualization
- A/B testing results
- Feature importance chart
- Purchase probability prediction tool

---
